# Mini Task Manager

A simple Python GUI application to manage tasks with priorities.

## Features
- Add tasks with a description and priority (1-5)
- Pop the highest priority task
- View tasks in priority order
- Interactive GUI with Tkinter

## How to Run
1. Make sure you have the newest version of Python to avoid any errors wihtin the program.
2. Open a terminal and run:

python gui.py
